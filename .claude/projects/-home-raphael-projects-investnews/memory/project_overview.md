InvestNews is a full-stack news management system for investors, built as a technical challenge for Investidor10.

**Why:** Demonstrate full-stack competence in the Laravel ecosystem with a single-deploy architecture.

**How to apply:** The project values single-deploy simplicity (React embedded via Vite), cookie-based auth (Sanctum SPA), and real-time features (Reverb WebSocket for market ticks). Live at https://ckao.in.
